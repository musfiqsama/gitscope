# 🚀 GitScope

> A powerful GitHub portfolio analyzer that transforms developer profiles into recruiter-friendly insights using custom scoring and interactive visualizations.

---

## ✨ Overview

**GitScope** is a frontend-driven analytics tool that fetches real-time GitHub data and converts it into structured insights about a developer’s profile, repositories, and activity.

It helps:

* 👨‍💻 Developers understand their portfolio strength
* 🧑‍💼 Recruiters quickly evaluate candidates
* 📊 Teams compare developer profiles visually

---

## 🔥 Features

* 🎯 **Portfolio Scoring System** (0–100)
* 📊 **Interactive Charts** (Language usage & activity trends)
* 🧠 **Recruiter Insights** (Strengths, weaknesses, role fit)
* 🔍 **Repository Health Analysis**
* 📄 **Profile README Preview**
* ⚖️ **Compare Two GitHub Profiles**
* 🔥 **Contribution Heatmap Visualization**
* 📦 **Top Repository Filtering & Search**
* 📥 **Export Report Option**

---

## 🧠 How It Works

GitScope uses a custom scoring algorithm based on:

* Profile completeness
* Repository quality
* Documentation (README, descriptions)
* Activity consistency
* Stars & forks (popularity signals)
* Tech stack clarity

All processing is done **client-side**, without any backend.

---

## 🛠️ Tech Stack

* ⚛️ React (CDN-based)
* 📊 Chart.js
* 📄 Marked (Markdown parser)
* 🎨 Custom CSS (Dark UI)
* 🌐 GitHub REST API

## 🚀 Getting Started

### Run Locally

```bash
git clone https://github.com/your-username/gitscope.git
cd gitscope
python -m http.server 5500
```

Then open:

```
http://localhost:5500
```
